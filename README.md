🍔 Krusty Krab Ordering System

## 📖 Deskripsi Proyek

Krusty Krab Ordering System merupakan aplikasi web sederhana yang dibuat untuk memenuhi tugas mata kuliah **Pemrograman Web 1**. Sistem ini memungkinkan pelanggan untuk melakukan pemesanan menu makanan dan merchandise secara online serta memudahkan pegawai dalam melihat daftar pesanan yang masuk.

Proyek ini menerapkan konsep dasar pengembangan web seperti:

- HTML
- CSS
- PHP
- MySQL
- Form Login dan Registrasi
- CRUD Data Pesanan
- Session Management

---

## 🎯 Tujuan Proyek

Tujuan pembuatan aplikasi ini adalah:

- Menerapkan konsep dasar pemrograman web.
- Membuat sistem pemesanan berbasis web.
- Mengimplementasikan autentikasi pengguna.
- Menampilkan dan mengelola data pesanan pelanggan.

---

## 🖥️ Tampilan Antarmuka (UI)

### 1. Homepage Customer

Halaman utama yang menampilkan informasi pengguna yang sedang login serta informasi toko.

**Fitur:**
- Menampilkan username customer.
- Menampilkan informasi Krusty Krab.
- Navigasi menuju halaman Orders.
- Tombol Logout.

---

### 2. Login Page

Halaman autentikasi pengguna untuk masuk ke dalam sistem.

**Fitur:**
- Input Username.
- Input Password.
- Validasi Login.
- Redirect ke halaman sesuai role pengguna.

---

### 3. Register Customer

Halaman pendaftaran akun customer baru.

**Fitur:**
- Registrasi Username.
- Registrasi Password.
- Penyimpanan data ke database.
- Redirect ke halaman login setelah berhasil mendaftar.

---

### 4. Orders Page

Halaman pemesanan produk Krusty Krab.

**Fitur:**
- Daftar Menu Makanan.
- Daftar Merchandise.
- Tabel Harga Produk.
- Form Pemesanan.
- Input Nama Customer.
- Input Nomor Telepon.
- Input Alamat Pengiriman.
- Tombol Submit Pesanan.

---

### 5. Homepage Employee

Halaman utama khusus pegawai.

**Fitur:**
- Menampilkan informasi akun pegawai.
- Akses ke daftar pesanan customer.
- Logout sistem.

---

### 6. Orders List

Halaman untuk melihat seluruh pesanan yang masuk.

**Fitur:**
- Daftar pesanan menu makanan.
- Daftar pesanan merchandise.
- Menampilkan:
  - Kode Produk
  - Nama Produk
  - Harga
  - Jumlah Pesanan
